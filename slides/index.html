Layouts and Includes

  • Defined in front matter
  • Loaded at _includes
  • Can be chained

---

title: Something
another: thing
--- 

Your MD file, Liquid, etc

Collections

  • Way to group files
  • Tag based
  • Can be dynamic

---
layout: somename
title: Something
tags: post
--- 

Your MD file, Liquid, etc
				

{% for post in collections.post %}
  use the post object here
{% endfor %}

Passthrough File Copy

  • If Eleventy doesn't recognize a file, it ignores it.
  • If Eleventy doesn't recognize a file, it ignores it.
  • You can tell Eleventy to copy a file or set of files

module.exports = function(eleventyConfig) {

	eleventyConfig.addPassthroughCopy("img");
	eleventyConfig.addPassthroughCopy("css");

};
